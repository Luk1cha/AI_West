## Project Status

This is a **completed Hackathon project** developed as a proof-of-concept. The team has successfully delivered:
- ✅ Functional mobile app (iOS/Android/Web)
- ✅ ESP32 microcontroller firmware
- ✅ Firebase cloud integration
- ✅ AI-driven watering system
- ✅ Community features (forum)

The project is published as-is for educational and reference purposes.

## Technologies Used

- **Frontend**: React Native, Expo, TypeScript
- **Backend**: Firebase, Express.js
- **Hardware**: ESP32, DHT sensors
- **Communication**: Bluetooth LE, REST API
- **Design**: NativeWind (Tailwind CSS)
